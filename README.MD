## 📖 Descripción

Este repositorio reúne los ejercicios, algoritmos y programas desarrollados durante la asignatura de **Métodos Numéricos** utilizando Python.

Se incluyen implementaciones prácticas de diversos métodos empleados para la resolución de problemas matemáticos mediante aproximaciones numéricas.

# 1️⃣ Tema 1: Errores Numéricos

Los errores numéricos aparecen debido a las limitaciones de representación de los números dentro de una computadora.

### Contenido

* Acumulación de errores
* Comparación incorrecta de decimales
* Overflow
* Pérdida de precisión por magnitud
* Tipos de error (absoluto, relativo y porcentual)
